+ <kbd>o</kbd> switch to the other end of selected text.
+ <kbd>w</kbd> move to next word
+ <kbd>y</kbd> copy
+ <kbd>p</kbd> paste
+ <kbd>G</kbd> to move to end of file, 
+ <kbd>gg</kbd> to move to beginning
+ <kbd>d</kbd> delete selected text
+ <kbd>dd</kbd> delete one line
+ <kbd>dw</kbd> delete current word
+ <kbd>></kbd> Indent right
+ <kbd><</kbd> Indent left
+ <kbd>qa</kbd> quit all files

```bash 
:sp filename.txt           " Horizontal split with existing file
:vsp filename.txt          " Vertical split with existing file
```

+ <kbd>Ctrl+w w</kbd> cycle through windows
